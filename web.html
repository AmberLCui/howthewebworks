1. What is HTTP? HTTP is how browsers and servers communicate with encryption. 2. What is a URL? A URL is an address used by browsers to retrieve resources on the web. 3. What is DNS? DNS takes a hostname and translate that into an IP Address. 4. What is a query string? A query string provides extra information and starts with a '?'. Is key value pairs. 5. What are two HTTP verbs and how are they different? The two HTTP verbs are GET and POST. GET are requests without side effects and POST requests with side effects. 6. What is an HTTP request? HTTP request is made by a client to a host in a server. 7. What is an HTTP response? An HTTP response is made by the server to a client. It provides the client with resource they requested. 8. What is an HTTP header? Give a couple examples of request and response headers you have seen. A header provides information about the request or response. Examples for request: Cookies, Cache Examples for response: Set Cookie, Content Type 9. What are the processes that happen when you type "http://somesite.com/some/page.html" into a browser? First, the browser will turn the name into an IP Address with DNS. Second, the browser will make a request to that IP Address. Third, the server sends a response. Fourth, the browser will creat a DOM from the HTML response, along with other resources. Fifth, the browser wil make seperate HTTP requests. Will also recieve responses from server.
